# 🌌 3D Solar System Explorer

An interactive 3D Solar System simulation built using Three.js and modern web technologies. This educational web application allows users to explore celestial bodies, visualize orbital mechanics, and learn astronomy concepts through immersive real-time interaction.

## Live Demo

https://3dsolar.gamer.free/

## Features

* Interactive 3D Solar System visualization
* Real-time orbital motion simulation
* Earth, Moon, and Mars exploration
* Educational astronomy content
* Dynamic planetary information panels
* Orbit visualization controls
* Adjustable simulation speed
* Planet focus and camera controls
* Responsive modern UI
* Three.js powered WebGL rendering

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Three.js
* WebGL
* Tailwind CSS

## Educational Concepts

* Orbital Mechanics
* Planetary Motion
* Kepler's Laws
* Relative Planetary Distances
* Rotation and Revolution
* Solar System Structure
